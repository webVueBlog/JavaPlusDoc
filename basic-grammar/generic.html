    зӮ№еҮ»еӢҳиҜҜissues (opens new window)пјҢе“Әеҗ’ж„ҹи°ўеӨ§е®¶зҡ„йҳ…иҜ»

    # ж·ұе…Ҙи§ЈжһҗJavaжіӣеһӢ

    Java еңЁ 1.5 ж—¶еўһеҠ дәҶжіӣеһӢжңәеҲ¶

    class Arraylist {
    private Object[] objs;
    private int i = 0;
    public void add(Object obj) {
        objs[i++] = obj;
    }
    
    public Object get(int i) {
        return objs[i];
    }
}
    1. Arraylist еҸҜд»Ҙеӯҳж”ҫд»»дҪ•зұ»еһӢзҡ„ж•°жҚ®пјҲж—ўеҸҜд»Ҙеӯҳеӯ—з¬ҰдёІпјҢд№ҹеҸҜд»Ҙж··е…Ҙж—ҘжңҹпјүпјҢеӣ дёәжүҖжңүзұ»йғҪз»§жүҝиҮӘ Object зұ»гҖӮ
    2. д»Һ Arraylist еҸ–еҮәж•°жҚ®зҡ„ж—¶еҖҷйңҖиҰҒејәеҲ¶зұ»еһӢиҪ¬жҚўпјҢеӣ дёәзј–иҜ‘еҷЁе№¶дёҚиғҪзЎ®е®ҡдҪ еҸ–зҡ„жҳҜеӯ—з¬ҰдёІиҝҳжҳҜж—ҘжңҹгҖӮ

    1пјүзұ»еһӢеҸӮж•°еҢ–

    жіӣеһӢзҡ„жң¬иҙЁжҳҜеҸӮж•°еҢ–зұ»еһӢпјҢд№ҹе°ұжҳҜиҜҙпјҢеңЁе®ҡд№үзұ»гҖҒжҺҘеҸЈжҲ–ж–№жі•ж—¶пјҢеҸҜд»ҘдҪҝз”ЁдёҖдёӘжҲ–еӨҡдёӘзұ»еһӢеҸӮж•°жқҘиЎЁзӨәеҸӮж•°еҢ–зұ»еһӢгҖӮ

    2пјүзұ»еһӢж“ҰйҷӨ

    еңЁ Java зҡ„жіӣеһӢжңәеҲ¶дёӯпјҢжңүдёӨдёӘйҮҚиҰҒзҡ„жҰӮеҝөпјҡзұ»еһӢж“ҰйҷӨе’ҢйҖҡй…Қз¬ҰгҖӮ

    жіӣеһӢеңЁзј–иҜ‘ж—¶дјҡе°ҶжіӣеһӢзұ»еһӢж“ҰйҷӨпјҢе°ҶжіӣеһӢзұ»еһӢжӣҝжҚўжҲҗ Object зұ»еһӢгҖӮиҝҷжҳҜдёәдәҶеҗ‘еҗҺе…је®№пјҢйҒҝе…ҚеҜ№еҺҹжңүзҡ„ Java д»Јз ҒйҖ жҲҗеҪұе“ҚгҖӮ

    3пјүйҖҡй…Қз¬Ұ

    йҖҡй…Қз¬Ұз”ЁдәҺиЎЁзӨәжҹҗз§ҚжңӘзҹҘзҡ„зұ»еһӢпјҢдҫӢеҰӮ List<?> иЎЁзӨәдёҖдёӘеҸҜд»ҘеӯҳеӮЁд»»дҪ•зұ»еһӢеҜ№иұЎзҡ„ ListпјҢдҪҶжҳҜдёҚиғҪеҜ№е…¶дёӯзҡ„е…ғзҙ иҝӣиЎҢж·»еҠ ж“ҚдҪңгҖӮйҖҡй…Қз¬ҰеҸҜд»Ҙз”ЁжқҘи§ЈеҶізұ»еһӢдёҚзЎ®е®ҡзҡ„жғ…еҶөпјҢдҫӢеҰӮеңЁж–№жі•еҸӮж•°жҲ–иҝ”еӣһеҖјдёӯдҪҝз”ЁгҖӮ
