    # TreeMapиҜҰи§Ј

    HashMap жҳҜж— еәҸзҡ„пјҢжүҖд»ҘжңүдәҶ LinkedHashMapпјҢеҠ дёҠдәҶеҸҢеҗ‘й“ҫиЎЁеҗҺпјҢе°ұеҸҜд»ҘдҝқжҢҒе…ғзҙ зҡ„жҸ’е…ҘйЎәеәҸе’Ңи®ҝй—®йЎәеәҸпјҢйӮЈ TreeMap е‘ўпјҹ

    TreeMap з”ұзәўй»‘ж ‘е®һзҺ°пјҢеҸҜд»ҘдҝқжҢҒе…ғзҙ зҡ„иҮӘз„¶йЎәеәҸпјҢжҲ–иҖ…е®һзҺ°дәҶ Comparator жҺҘеҸЈзҡ„иҮӘе®ҡд№үйЎәеәҸгҖӮ

    зәўй»‘ж ‘пјҲиӢұиҜӯпјҡRedвҖ“black treeпјүжҳҜдёҖз§ҚиҮӘе№іиЎЎзҡ„дәҢеҸүжҹҘжүҫж ‘пјҲBinary Search TreeпјүпјҢз»“жһ„еӨҚжқӮпјҢдҪҶеҚҙжңүзқҖиүҜеҘҪзҡ„жҖ§иғҪпјҢе®ҢжҲҗжҹҘжүҫгҖҒжҸ’е…Ҙе’ҢеҲ йҷӨзҡ„ж—¶й—ҙеӨҚжқӮеәҰеқҮдёә log(n)гҖӮ

    дәҢеҸүжҹҘжүҫж ‘жҳҜдёҖз§Қеёёи§Ғзҡ„ж ‘еҪўз»“жһ„пјҢе®ғзҡ„жҜҸдёӘиҠӮзӮ№йғҪеҢ…еҗ«дёҖдёӘй”®еҖјеҜ№гҖӮжҜҸдёӘиҠӮзӮ№зҡ„е·Ұеӯҗж ‘иҠӮзӮ№зҡ„й”®еҖје°ҸдәҺиҜҘиҠӮзӮ№зҡ„й”®еҖјпјҢеҸіеӯҗж ‘иҠӮзӮ№зҡ„й”®еҖјеӨ§дәҺиҜҘиҠӮзӮ№зҡ„й”®еҖјпјҢиҝҷдёӘзү№жҖ§дҪҝеҫ—дәҢеҸүжҹҘжүҫж ‘йқһеёёйҖӮеҗҲиҝӣиЎҢж•°жҚ®зҡ„жҹҘжүҫе’ҢжҺ’еәҸж“ҚдҪңгҖӮ

    дёҖйў—е…ёеһӢзҡ„дәҢеҸүжҹҘжүҫж ‘пјҡ

    • 1пјүе·Ұеӯҗж ‘дёҠжүҖжңүиҠӮзӮ№зҡ„еҖјеқҮе°ҸдәҺжҲ–зӯүдәҺе®ғзҡ„ж №з»“зӮ№зҡ„еҖјгҖӮ
    • 2пјүеҸіеӯҗж ‘дёҠжүҖжңүиҠӮзӮ№зҡ„еҖјеқҮеӨ§дәҺжҲ–зӯүдәҺе®ғзҡ„ж №з»“зӮ№зҡ„еҖјгҖӮ
    • 3пјүе·ҰгҖҒеҸіеӯҗж ‘д№ҹеҲҶеҲ«дёәдәҢеҸүжҹҘжүҫж ‘гҖӮ

    дәҢеҸүжҹҘжүҫж ‘з”ЁжқҘжҹҘжүҫйқһеёёж–№йқўпјҢд»Һж №иҠӮзӮ№ејҖе§ӢйҒҚеҺҶпјҢеҰӮжһңеҪ“еүҚиҠӮзӮ№зҡ„й”®еҖјзӯүдәҺиҰҒжҹҘжүҫзҡ„й”®еҖјпјҢеҲҷжҹҘжүҫжҲҗеҠҹпјӣеҰӮжһңиҰҒжҹҘжүҫзҡ„й”®еҖје°ҸдәҺеҪ“еүҚиҠӮзӮ№зҡ„й”®еҖјпјҢеҲҷз»§з»ӯйҒҚеҺҶе·Ұеӯҗж ‘пјӣеҰӮжһңиҰҒжҹҘжүҫзҡ„й”®еҖјеӨ§дәҺеҪ“еүҚиҠӮзӮ№зҡ„й”®еҖјпјҢеҲҷз»§з»ӯйҒҚеҺҶеҸіеӯҗж ‘гҖӮеҰӮжһңйҒҚеҺҶеҲ°еҸ¶еӯҗиҠӮзӮ№д»Қз„¶жІЎжңүжүҫеҲ°пјҢеҲҷжҹҘжүҫеӨұиҙҘгҖӮ

    жҸ’е…Ҙж“ҚдҪңд№ҹйқһеёёз®ҖеҚ•пјҢд»Һж №иҠӮзӮ№ејҖе§ӢйҒҚеҺҶпјҢеҰӮжһңиҰҒжҸ’е…Ҙзҡ„й”®еҖје°ҸдәҺеҪ“еүҚиҠӮзӮ№зҡ„й”®еҖјпјҢеҲҷе°Ҷе…¶жҸ’е…ҘеҲ°е·Ұеӯҗж ‘дёӯпјӣеҰӮжһңиҰҒжҸ’е…Ҙзҡ„й”®еҖјеӨ§дәҺеҪ“еүҚиҠӮзӮ№зҡ„й”®еҖјпјҢеҲҷе°Ҷе…¶жҸ’е…ҘеҲ°еҸіеӯҗж ‘дёӯгҖӮеҰӮжһңиҰҒжҸ’е…Ҙзҡ„й”®еҖје·Із»ҸеӯҳеңЁдәҺж ‘дёӯпјҢеҲҷжӣҙж–°иҜҘиҠӮзӮ№зҡ„еҖјгҖӮ

    еҲ йҷӨж“ҚдҪңзЁҚеҫ®еӨҚжқӮдёҖдәӣпјҢйңҖиҰҒиҖғиҷ‘еӨҡз§Қжғ…еҶөпјҢеҢ…жӢ¬иҰҒеҲ йҷӨзҡ„иҠӮзӮ№жҳҜеҸ¶еӯҗиҠӮзӮ№гҖҒиҰҒеҲ йҷӨзҡ„иҠӮзӮ№еҸӘжңүдёҖдёӘеӯҗиҠӮзӮ№гҖҒиҰҒеҲ йҷӨзҡ„иҠӮзӮ№жңүдёӨдёӘеӯҗиҠӮзӮ№зӯүзӯүгҖӮ

    жҖ»д№ӢпјҢдәҢеҸүжҹҘжүҫж ‘жҳҜдёҖз§Қйқһеёёеёёз”Ёзҡ„ж•°жҚ®з»“жһ„пјҢе®ғеҸҜд»Ҙеё®еҠ©жҲ‘д»¬е®һзҺ°ж•°жҚ®зҡ„жҹҘжүҫгҖҒжҺ’еәҸе’ҢеҲ йҷӨзӯүж“ҚдҪңгҖӮ

    е№іиЎЎдәҢеҸүж ‘е°ұеғҸжҳҜдёҖжЈөж ‘еҪўз§ӨпјҢе®ғзҡ„е·ҰеҸідёӨиҫ№зҡ„йҮҚйҮҸиҰҒе°ҪеҸҜиғҪзҡ„е№іиЎЎгҖӮеҪ“жҲ‘д»¬еҫҖе№іиЎЎдәҢеҸүж ‘дёӯжҸ’е…ҘдёҖдёӘиҠӮзӮ№ж—¶пјҢе№іиЎЎдәҢеҸүж ‘дјҡиҮӘеҠЁи°ғж•ҙиҠӮзӮ№зҡ„дҪҚзҪ®пјҢд»ҘдҝқиҜҒж ‘зҡ„е·ҰеҸідёӨиҫ№зҡ„й«ҳеәҰе·®дёҚи¶…иҝҮ1гҖӮзұ»дјјең°пјҢеҪ“жҲ‘д»¬еҲ йҷӨдёҖдёӘиҠӮзӮ№ж—¶пјҢе№іиЎЎдәҢеҸүж ‘д№ҹдјҡиҮӘеҠЁи°ғж•ҙиҠӮзӮ№зҡ„дҪҚзҪ®пјҢд»ҘдҝқиҜҒж ‘зҡ„е·ҰеҸідёӨиҫ№зҡ„й«ҳеәҰе·®дёҚи¶…иҝҮ1гҖӮ

    еёёи§Ғзҡ„е№іиЎЎдәҢеҸүж ‘еҢ…жӢ¬AVLж ‘гҖҒзәўй»‘ж ‘зӯүзӯүпјҢе®ғд»¬йғҪжҳҜйҖҡиҝҮж—ӢиҪ¬ж“ҚдҪңжқҘи°ғж•ҙж ‘зҡ„е№іиЎЎпјҢдҪҝеҫ—е·Ұеӯҗж ‘е’ҢеҸіеӯҗж ‘зҡ„й«ҳеәҰе°ҪеҸҜиғҪжҺҘиҝ‘гҖӮ

    дҫӢеҰӮпјҢеҜ№дәҺдёҖдёӘйңҖиҰҒйў‘з№ҒиҝӣиЎҢжҹҘжүҫж“ҚдҪңзҡ„еңәжҷҜпјҢеҰӮеӯ—е…ёж ‘гҖҒе“ҲеёҢиЎЁзӯүж•°жҚ®з»“жһ„пјҢеҸҜд»ҘдҪҝз”ЁAVLж ‘жқҘиҝӣиЎҢдјҳеҢ–гҖӮеҸҰеӨ–пјҢAVLж ‘д№ҹйҖӮз”ЁдәҺйңҖиҰҒдҝқиҜҒж•°жҚ®жңүеәҸжҖ§зҡ„еңәжҷҜпјҢеҰӮж•°жҚ®еә“дёӯзҡ„зҙўеј•гҖӮ

    AVLж ‘зҡ„еҸ‘жҳҺеҜ№и®Ўз®—жңәз§‘еӯҰзҡ„еҸ‘еұ•жңүзқҖйҮҚиҰҒзҡ„еҪұе“ҚпјҢдёҚд»…дёәеҗҺжқҘзҡ„е№іиЎЎдәҢеҸүж ‘жҸҗдҫӣдәҶеҹәзЎҖпјҢиҖҢдё”дёәе…¶д»–йўҶеҹҹзҡ„ж•°жҚ®з»“жһ„е’Ңз®—жі•жҸҗдҫӣдәҶеҗҜзӨәгҖӮ

    # зәўй»‘ж ‘

    зәўй»‘ж ‘пјҢйЎҫеҗҚжҖқд№үпјҢе°ұжҳҜиҠӮзӮ№жҳҜзәўиүІжҲ–иҖ…й»‘иүІзҡ„е№іиЎЎдәҢеҸүж ‘пјҢе®ғйҖҡиҝҮйўңиүІзҡ„зәҰжқҹжқҘз»ҙжҢҒдәҢеҸүж ‘зҡ„е№іиЎЎпјҢе®ғиҰҒжұӮд»»ж„ҸдёҖжқЎи·Ҝеҫ„дёҠзҡ„й»‘иүІиҠӮзӮ№ж•°зӣ®зӣёеҗҢпјҢеҗҢж—¶иҝҳйңҖиҰҒж»Ўи¶ідёҖдәӣе…¶д»–зү№е®ҡзҡ„жқЎд»¶пјҢеҰӮзәўиүІиҠӮзӮ№зҡ„зҲ¶иҠӮзӮ№еҝ…йЎ»дёәй»‘иүІиҠӮзӮ№зӯүгҖӮ

    • 1пјүжҜҸдёӘиҠӮзӮ№йғҪеҸӘиғҪжҳҜзәўиүІжҲ–иҖ…й»‘иүІ
    • 2пјүж №иҠӮзӮ№жҳҜй»‘иүІ
    • 3пјүжҜҸдёӘеҸ¶иҠӮзӮ№пјҲNIL иҠӮзӮ№пјҢз©әиҠӮзӮ№пјүжҳҜй»‘иүІзҡ„гҖӮ
    • 4пјүеҰӮжһңдёҖдёӘиҠӮзӮ№жҳҜзәўиүІзҡ„пјҢеҲҷе®ғдёӨдёӘеӯҗиҠӮзӮ№йғҪжҳҜй»‘иүІзҡ„гҖӮд№ҹе°ұжҳҜиҜҙеңЁдёҖжқЎи·Ҝеҫ„дёҠдёҚиғҪеҮәзҺ°зӣёйӮ»зҡ„дёӨдёӘзәўиүІиҠӮзӮ№гҖӮ
    • 5пјүд»Һд»»дёҖиҠӮзӮ№еҲ°е…¶жҜҸдёӘеҸ¶еӯҗзҡ„жүҖжңүи·Ҝеҫ„йғҪеҢ…еҗ«зӣёеҗҢж•°зӣ®зҡ„й»‘иүІиҠӮзӮ№гҖӮ

    з”ұдәҺзәўй»‘ж ‘зҡ„е№іиЎЎеәҰжҜ”AVLж ‘зЁҚдҪҺпјҢеӣ жӯӨеңЁиҝӣиЎҢжҸ’е…Ҙе’ҢеҲ йҷӨж“ҚдҪңж—¶йңҖиҰҒиҝӣиЎҢзҡ„ж—ӢиҪ¬ж“ҚдҪңиҫғе°‘пјҢдҪҶжҳҜеңЁжҹҘжүҫж“ҚдҪңж—¶ж•ҲзҺҮд»Қз„¶иҫғй«ҳгҖӮзәўй»‘ж ‘йҖӮз”ЁдәҺиҜ»еҶҷж“ҚдҪңжҜ”иҫғеқҮиЎЎзҡ„еңәжҷҜгҖӮ

    # иҮӘз„¶йЎәеәҸ

    й»ҳи®Өжғ…еҶөдёӢпјҢTreeMap жҳҜж №жҚ® key зҡ„иҮӘз„¶йЎәеәҸжҺ’еҲ—зҡ„гҖӮжҜ”еҰӮиҜҙж•ҙж•°пјҢе°ұжҳҜеҚҮеәҸпјҢ1гҖҒ2гҖҒ3гҖҒ4гҖҒ5гҖӮ

    # иҮӘе®ҡд№үжҺ’еәҸ

    еҰӮжһңиҮӘз„¶йЎәеәҸдёҚж»Ўи¶іпјҢйӮЈе°ұеҸҜд»ҘеңЁеЈ°жҳҺ TreeMap еҜ№иұЎзҡ„ж—¶еҖҷжҢҮе®ҡжҺ’еәҸи§„еҲҷгҖӮ

    Comparator.reverseOrder() иҝ”еӣһзҡ„жҳҜ Collections.ReverseComparator еҜ№иұЎпјҢе°ұжҳҜз”ЁжқҘеҸҚиҪ¬йЎәеәҸзҡ„пјҢйқһеёёж–№дҫҝгҖӮ

    HashMap жҳҜж— еәҸзҡ„пјҢжҸ’е…Ҙзҡ„йЎәеәҸйҡҸзқҖе…ғзҙ зҡ„еўһеҠ дјҡдёҚеҒңең°еҸҳеҠЁгҖӮдҪҶ TreeMap иғҪеӨҹиҮіе§ӢиҮіз»ҲжҢүз…§жҢҮе®ҡзҡ„йЎәеәҸжҺ’еҲ—пјҢиҝҷеҜ№дәҺйңҖиҰҒиҮӘе®ҡд№үжҺ’еәҸзҡ„еңәжҷҜ

    # жҺ’еәҸзҡ„еҘҪеӨ„

    ж—ўз„¶ TreeMap зҡ„е…ғзҙ жҳҜз»ҸиҝҮжҺ’еәҸзҡ„пјҢйӮЈжүҫеҮәжңҖеӨ§зҡ„йӮЈдёӘпјҢжңҖе°Ҹзҡ„йӮЈдёӘпјҢжҲ–иҖ…жүҫеҮәжүҖжңүеӨ§дәҺжҲ–иҖ…е°ҸдәҺжҹҗдёӘеҖјзҡ„й”®жқҘиҜҙпјҢе°ұж–№дҫҝеӨҡдәҶгҖӮ

    TreeMap иҖғиҷ‘еҫ—еҫҲе‘Ёе…ЁпјҢжҒ°еҘҪе°ұжҸҗдҫӣдәҶ lastKey()гҖҒfirstKey() иҝҷж ·иҺ·еҸ–жңҖеҗҺдёҖдёӘ key е’Ңз¬¬дёҖдёӘ key зҡ„ж–№жі•гҖӮ

    headMap() иҺ·еҸ–зҡ„жҳҜеҲ°жҢҮе®ҡ key д№ӢеүҚзҡ„ keyпјӣtailMap() иҺ·еҸ–зҡ„жҳҜжҢҮе®ҡ key д№ӢеҗҺзҡ„ keyпјҲеҢ…жӢ¬жҢҮе®ҡ keyпјүгҖӮ

    # еҰӮдҪ•йҖүжӢ© Map

    йңҖиҰҒиҖғиҷ‘д»ҘдёӢеӣ зҙ пјҡ

    1. жҳҜеҗҰйңҖиҰҒжҢүз…§й”®зҡ„иҮӘз„¶йЎәеәҸжҲ–иҖ…иҮӘе®ҡд№үйЎәеәҸиҝӣиЎҢжҺ’еәҸгҖӮеҰӮжһңйңҖиҰҒжҢүз…§й”®жҺ’еәҸпјҢеҲҷеҸҜд»ҘдҪҝз”Ё TreeMapпјӣеҰӮжһңдёҚйңҖиҰҒжҺ’еәҸпјҢеҲҷеҸҜд»ҘдҪҝз”Ё HashMap жҲ– LinkedHashMapгҖӮ
    2. жҳҜеҗҰйңҖиҰҒдҝқжҢҒжҸ’е…ҘйЎәеәҸгҖӮеҰӮжһңйңҖиҰҒдҝқжҢҒжҸ’е…ҘйЎәеәҸпјҢеҲҷеҸҜд»ҘдҪҝз”Ё LinkedHashMapпјӣеҰӮжһңдёҚйңҖиҰҒдҝқжҢҒжҸ’е…ҘйЎәеәҸпјҢеҲҷеҸҜд»ҘдҪҝз”Ё TreeMap жҲ– HashMapгҖӮ
    3. жҳҜеҗҰйңҖиҰҒй«ҳж•Ҳзҡ„жҹҘжүҫгҖӮеҰӮжһңйңҖиҰҒй«ҳж•Ҳзҡ„жҹҘжүҫпјҢеҲҷеҸҜд»ҘдҪҝз”Ё LinkedHashMap жҲ– HashMapпјҢеӣ дёәе®ғд»¬зҡ„жҹҘжүҫж“ҚдҪңзҡ„ж—¶й—ҙеӨҚжқӮеәҰдёә O(1)пјҢиҖҢжҳҜ TreeMap жҳҜ O(log n)гҖӮ

    LinkedHashMap еҶ…йғЁдҪҝз”Ёе“ҲеёҢиЎЁжқҘеӯҳеӮЁй”®еҖјеҜ№пјҢе№¶дҪҝз”ЁдёҖдёӘеҸҢеҗ‘й“ҫиЎЁжқҘз»ҙжҠӨжҸ’е…ҘйЎәеәҸпјҢдҪҶжҹҘжүҫж“ҚдҪңеҸӘйңҖиҰҒеңЁе“ҲеёҢиЎЁдёӯиҝӣиЎҢпјҢдёҺй“ҫиЎЁж— е…іпјҢжүҖд»Ҙж—¶й—ҙеӨҚжқӮеәҰдёә O(1)
